# adakami-baru

Aplikasi React/Vite untuk halaman pusat bantuan pelanggan.

## Isolasi repository

Repository ini tidak menggunakan submodule Git dan tidak menyimpan konfigurasi remote Git di dalam source code. Jika repository ini perlu dipublikasikan, tambahkan remote secara eksplisit hanya pada clone yang digunakan.

Aset statis aplikasi disimpan di folder `public/` agar build tidak bergantung pada folder atau repository lain.

## Perintah

- `npm run dev` menjalankan server pengembangan.
- `npm run build` membuat build produksi.
- `npm run preview` menjalankan pratinjau build produksi.
